### Running with Docker

#### Cloning the repositorie

```git clone https://github.com/b3coded/calangos-hunt```

```cd calangos-hunt```


#### Build image

```docker build -t calangoshunt .```

#### Running container

```docker run -d calangoshunt -p 8080:80```

Open your browser at http://localhost:8080/ and enjoy!
